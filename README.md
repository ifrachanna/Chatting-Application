#  Chatting Application (Java Desktop)

A simple **Client-Server Chatting Application** developed using **Java (Swing + Socket Programming)** that enables real-time communication between users over a network.

---

##  Overview
This project demonstrates how to build a **real-time messaging system** using Java networking concepts. It allows multiple users to communicate by connecting to a central server.

The application is based on the **Client-Server Architecture** and uses **TCP/UDP socket programming** for communication.

---

##  Features
- 🔹 Real-time text messaging  
- 🔹 Client-Server communication  
- 🔹 Simple and user-friendly GUI (Java Swing)  
- 🔹 Message sending and receiving system  
- 🔹 Lightweight and easy to run  

---

##  Technologies Used
- **Programming Language:** Java  
- **GUI:** Java Swing (AWT & JFC)  
- **Networking:** Socket Programming (java.net, java.io)  
- **Concepts:** Client-Server Model, TCP/UDP  

---

##  How It Works
- The **Server** starts first and waits for client connections  
- The **Client** connects to the server using IP & port  
- Once connected, both can send and receive messages  
- Communication happens through **sockets**  

---

##  System Architecture
- Server Module  
- Client Module  

Both modules communicate using a **dedicated socket connection**

---

##  How to Run

### 1. Run Server
- Compile and run the server file  
- Server starts listening on a specific port  

### 2. Run Client
- Compile and run client file  
- Enter server IP (e.g., localhost) and port  

### 3. Start Chatting 🎉
- Send messages between client and server
